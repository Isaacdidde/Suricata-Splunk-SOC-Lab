## Overview
This repository documents a hands-on SOC homelab built to understand
network traffic analysis, intrusion detection and prevention, log monitoring,
and automation using Bash scripting.

## Tools & Technologies
- Suricata (IDS & IPS)
- Wireshark
- Splunk
- Bash scripting
- Linux (Kali)

## Lab Architecture
- Traffic captured and analyzed using Wireshark
- Network threats detected and blocked using Suricata IDS/IPS
- Logs forwarded and analyzed in Splunk
- Automation implemented using Bash scripts

## Features Implemented
- Suricata IDS and IPS configuration
- Custom rule testing and alert generation
- eve.json structured log analysis
- Splunk log ingestion and dashboards
- Bash scripts for automation and monitoring
